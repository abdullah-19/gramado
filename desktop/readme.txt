
 Gramado Desktop

 Apps for User environment.


  /apps 
      aplicativos plenos com diversos propósitos.

  /core 
      Conjunto dos aplicativos básicos que podem ser usados na inicialização.
      Na configuração de um sistema mínimo, é daqui que deve sair os
      programas em user mode.
      São os programas mais básicos que tem quando se deseja configurar 
      um sistema sem muitos aplicativos.
      aqui não deve ficar nenhuma ferramenta, utilitário ou aplicativo grande,
      só aplicativos relacionados com a inicialziação de teste de uma configuração 
      de sistema mínimo.

  /tests
       Aplicativos usados para testar o desempenho do sistema
       Aplicativos usados pelo desenvolvedor do sistema para testar 
       funcionalidades e aprimorar rotinas.
       Aplicativos em fase embrionária, qua ainda não atingiram o status de
       aplicativo pleno.

   /tool
       Aflicativos que são ferramentas do desenvolvedor.
       Ficam aqui os editores de texto, os compiladores, os
       interpretadores e outros.
