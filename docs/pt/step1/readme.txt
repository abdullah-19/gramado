

   /step1

   Essa passo da documentação comtém somente as quatro primeiras fases que passei
durante o desenvolvimento do sistema. São elas:

  1) /step1/boot
  2) /step1/kernel
  3) /step1/init
  4) /step1/desktop 


=============================
  1) /step1/boot
      Sobre os elementos de inicialização do sistema,
até a parte onde o comando é passado para o kernel base.


=============================
  2) /step1/kernel
   Sobre o kernel base. kernel.bin.


=============================
  3) /step1/init

   Referente ao processo de inicialização, igual acontece 
nos sistemas unix-like.
   No Gramado foi criado um ambiente chamado Gramado Core,
que comtém três programas na inicialização, init.bin, shell.bin e taskman.bin.
Eles serão explicados nessa pasta. 


=============================
  4) /step1/desktop 
  Sobre os principais programas que rodam no desktop,
como aplicativos de teste do sistema e os primeiros aplicativos criados.
