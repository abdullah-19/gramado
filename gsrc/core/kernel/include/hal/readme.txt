Sobre o diretório:
=================
    
    Headers para o módulo hal.
    
    Uma característica do hal é oferecer abstração
    de hardware escondendo particularidades de hardware
    atravéz de abstração e encapsulamento.
    Portanto muitas variáveis devem ser escondidas nesse módulo.

    Aqui nesse diretório tem header para
    drivers de dispositivo.

    Os parâmetros aqui são gerais e pertencem a muitos modelos
    de dispositivos.

    As estruturas que devem aparecer aqui são aquelas que identificam o modelo
    de dispositivo, o nome do driver, o pathname do driver....

    @todo: todo header deve ter no começo as
       variáveis para handler address, driver name e driver path name.
