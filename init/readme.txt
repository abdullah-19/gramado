
   init/gxs

   Gramado X Server - GXS.
   
   
       Três programas que formam um conjunto de funcionalidades básicas em user mode
	   para o Gramado. Eles compartilham a memória virtual por enquanto.
	   
	   
	   Isso não é um servidor instalável, são recursos persistentes do sistema gramado.
	   
	   O gramado poderá instalar recursos gráficos em drivers e servidores, mas esse
	   será a opção mais básica disponível.
	   
	   
	   gxs init  
	   gxs shell
	   gxs taskman
	   
	   Gramado X Server Init
	   Gramado X Server Shell
	   Gramado X Server Taskman
